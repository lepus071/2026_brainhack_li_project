# Preliminary Results (N = 10)

Pilot outputs from the Hybrid Dynamic Neural Decoding Pipeline (see [`/README.md`](../README.md) and [`codes/Hybrid_Dynamic_Decoding_Pipeline.md`](../codes/Hybrid_Dynamic_Decoding_Pipeline.md) for methods).

- **`ml_results/`** — Per-subject and group-level VPA $R^2$ values (`.npy`) for all 7 variance-partitioning components (Unique_e/xf/xs, Shared_*).
- **`html_views/`** — Interactive glass-brain HTML visualizations (Phase 8A) of the group-level VPA maps.
- **`nifti_axes/`** — Group-level VPA NIfTI maps, AFNI group statistic volumes (`GroupStat_*+tlrc`), per-subject VPA NIfTI maps, and the behavioral covariates file used for group statistics (Phase 7).
- **`surfice_renders/`** — SurfIce macro script (`.gls`) for high-resolution surface rendering (Phase 8B).

> Note: these are pilot results from N = 10 subjects and are intended for internal review, not as final published findings.
