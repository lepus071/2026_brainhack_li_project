# `codes/` Index

This directory contains three versions of the analysis pipeline:

- **[`version_2/`](version_2/) — current pipeline.** Self-contained, 2-variable
  ($x_f$, $x_s$) dual-rate model, Phase 0A-7 plus follow-up analyses. Start
  here. See [`version_2/README.md`](version_2/README.md) for details and
  results.
- **[`version_0/`](version_0/) — original shared pipeline (archived).**
  Phase 0A-3, 7 as first written, including the original 7-component
  ($e$, $x_f$, $x_s$) framing. Superseded by `version_2/`.
- **[`version_1/`](version_1/) — exploratory 3-variable attempt (archived).**
  Adds the error signal $e$ and a Chein & Schneider (2012) interpretation.
  **Not adopted** for the main analysis — see the appendix of
  [`../Pilot_Results_Public_v2.md`](../Pilot_Results_Public_v2.md) for why.

See the [root README](../README.md) for the full project overview and results.
